# Product Catalog App

Android-приложение для отображения каталога продуктов с поиском и фильтрацией по категориям.

API - https://fakestoreapi.com/

## Стек
- Kotlin
- Coroutines
- Jetpack Compose
- Retrofit
- Room
- Hilt (DI)
- MVVM

## Модули
- **app** — UI и навигация  
- **domain** — бизнес-логика, use case  
- **data** — репозитории, источники данных
